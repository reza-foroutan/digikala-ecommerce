# Digikala E-commerce Project

## Description

A web development project replicating key features of Digikala's e-commerce platform using HTML5 and CSS3 with Visual Studio Code.

## Technologies

- HTML5
- CSS3
- Visual Studio Code

## Installation

1. Clone the repository
2. Open the project in Visual Studio Code
3. Run the project in your preferred web browser

## Features

- Responsive design
- Product catalog
- Navigation menu
- Shopping cart interface
